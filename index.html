Навыки

Каждый начинающий frontend-разработчик сегодня обязан располагать знаниями и опытом использования множества технологий и программ. Для качественного результата и работы в команде, я постоянно стремлюсь к получению новых знаний, изучению технологий и методик, а также развитию soft skills.

  • HTML5

    Применяю семантическую, адаптивную, кроссбраузерную верстку, с использованием методологии БЭМ. Пишу валидный переиспользуемый код, имею практический опыт работы с шаблонизатором HTML - handlebars.

  • CSS3

    В верстке использую Flexbox/Grid. Могу задействовать некоторые сторонние библиотеки, например, Bootstrap или animate.js.

  • JavaScript (ES6)

    Базовые знания Javascript, понимание основ ООП, взаимодействие с DOM и BOM -элементами. Руководствуясь принципами KISS и DRY, стараюсь поддерживать простой, понятный, поддерживаемый код, без излишнего дублирования и усложненных конструкций.

  • GitHub

    У каждого разработчика, особенно при командной работе, есть необходимость использовать Git. Вы пожете посмотреть мой профиль по ссылке.

  • Gulp

    В работе использую Gulp, также имел небольшой опыт со сборщиком проекта Webpack.

  • PostCSS

    Использование препроцессоров ускоряет разработку и упрощяет читабельность кода. PostCSS- современная технология, одна из самых часто загружаемых с npm инструментов для работы с CSS для многих разработчиков.

  • SASS/SCSS

    Имел также небольшой опыт работы с SASS/SCSS. Удобный, понятный и широко используемый инструмент.

  • Sketch

    Иногда возникает надобность что-то быстро подкорректировать в графическом редакторе. Photoshop теряет свою популярность, поэтому не лишним владеть инструментарием такой программы как Sketch.
